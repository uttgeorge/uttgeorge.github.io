Data visualization and Analysis Task

Covid Tracking Web-app

www.covidmonitor.cc
This is a real-time data visualization flask web-app focused on COVID-19 cases in USA. Most of the data were collected via API or web scraper from CDC, WHO and John Hopkins University. This is mainly a visualization project which is made by Python, JavaScript, html, css.
This project is also embeded a LSTM model to predict trend of USA deaths. The prediction is updated daily based on previous 100 days' data, and stored in a Azure SQL database.
Check the analysis at jupyter notebook or the whole project at github.
